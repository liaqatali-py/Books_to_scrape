# Books_to_scrape

This project is a web scraper for the website **Books to Scrape**.
It extracts book information such as title, price, stock availability, and rating.

## Features
- Scrapes book titles
- Extracts prices
- Checks availability
- Saves data to CSV

## Requirements
- Python 3.x
- Requests
